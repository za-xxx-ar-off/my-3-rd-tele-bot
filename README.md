Telegram Survey Bot — Phu Quoc Tours
Telegram-бот для проведения опроса среди туристов на острове Фукуок (Вьетнам).
Бот задаёт вопросы о посещённых экскурсиях и турах и сохраняет ответы пользователей в Google Sheets.
Проект предназначен для сбора статистики о популярности экскурсий среди туристов.
Основные функции
запуск опроса через /start
последовательная выдача вопросов
поддержка изображений экскурсий
выбор ответа через inline-кнопки
сохранение результатов в Google Sheets
возможность пройти опрос заново
поддержка Webhook (FastAPI)
Варианты ответов
Каждый вопрос имеет 4 варианта ответа:
✅ Был
❌ Не был
⭐ Хочу побывать
⏭ Пропустить
Ответы сохраняются в таблицу.
Структура Google Sheets
Используется одна таблица.
Название таблицы:
бот фукуок вьетнам
Структура листа
A	B	C	D+
Фото	Текст вопроса	(пусто)	ответы пользователей
Колонки:
A — ссылка на изображение экскурсии (Google Drive)
B — текст вопроса
D и далее — ответы пользователей
Первая строка
Первая строка — заголовки:
Фото | Текст | | @username1 | @username2 | ...
Каждый пользователь получает свою колонку.
Логика работы
Пользователь вводит /start
Бот ищет первый вопрос в таблице
Отправляет:
фото экскурсии (если есть)
текст
кнопки ответа
Пользователь выбирает ответ
Ответ записывается в Google Sheets
Бот отправляет следующий вопрос
После последнего вопроса предлагается начать заново

Переменные окружения
Необходимо задать:
BOT_TOKEN
RENDER_EXTERNAL_HOSTNAME
GOOGLE_CREDS_JSON
Токен Telegram-бота от BotFather.
RENDER_EXTERNAL_HOSTNAME
Хост приложения на Render.
Пример:
my-bot.onrender.com
GOOGLE_CREDS_JSON
JSON сервисного аккаунта Google.
Формат:
{
  "type": "service_account",
  "project_id": "...",
  "private_key_id": "...",
  "private_key": "...",
  "client_email": "...",
  ...
}
Подготовка Google Sheets
Создать таблицу
бот фукуок вьетнам
Добавить колонки:
A — фото
B — текст вопроса
Фото должны быть в Google Drive.
Пример ссылки:
https://drive.google.com/file/d/FILE_ID/view
Бот автоматически конвертирует её в прямую ссылку.
Установка зависимостей
pip install fastapi
pip install python-telegram-bot
pip install gspread
pip install google-auth
pip install uvicorn
или
pip install -r requirements.txt
Запуск локально
uvicorn main:app --reload
Webhook
Webhook автоматически устанавливается при запуске:
https://YOUR_HOST/webhook
Endpoints
Webhook
POST /webhook
Получает обновления Telegram.
Ping
GET /ping
Используется для проверки работы сервера.
Ответ:
OK
Архитектура кода
Основные части:
Handlers
/start
начинает опрос
CallbackQueryHandler
обрабатывает ответы кнопок
Работа с Google Sheets
Функции:
get_user_column()
находит колонку пользователя или создаёт новую
find_next_question_row()
находит следующий вопрос
send_question()
отправляет фото и текст вопроса
Поведение ответов
Если пользователь отвечает повторно:
ответ перезаписывается
Это позволяет менять выбор.
Перезапуск опроса
После последнего вопроса:
🔄 Начать заново
Опрос запускается с первого вопроса.
Логирование
Используется стандартный logging:
logging.INFO
Логи отображают:
запуск приложения
установку webhook
подключение Google Sheets
ошибки
Безопасность
Доступ к Google Sheets имеют только:
разработчик
заказчик
сервисный аккаунт
Пользователи Telegram не имеют доступа к таблице.
